---
title: "Commands"
description: "Command reference for the full hiring loop"
---

Commands follow `staplehire <resource> <action>`. Add **`--json`** when piping to an agent or script.

<Tip>
  Run `staplehire --help` or `staplehire <resource> --help` for the command tree for your installed version.
</Tip>

## Source

```bash
staplehire login
staplehire roles create --jd @senior-sdr.md
staplehire sourcing start <roleId>
staplehire sourcing prospects <roleId> --json
```

## Agent inbox

```bash
staplehire roles list --json
staplehire inbox messages <roleId> --json
staplehire inbox triage <roleId> --surface-above 7.5
```

Role inboxes use `hire+<role>@in.staplehire.com`.

## Investigate

```bash
staplehire candidates enrich <candidateId>
staplehire candidates research <candidateId>
staplehire candidates screening-result <candidateId> --json
```

## Interview

```bash
staplehire designs create <roleId> --brief "SDR screen"
staplehire candidates send-interview <candidateId> --design <designId>
staplehire sessions list <roleId> --json
```

## Hire

```bash
staplehire candidates send-email <candidateId> \
  --subject "Staplehire" \
  --title "Quick intro" \
  --body @message.md
```

## JSON output

```bash
staplehire sourcing prospects rol_abc123 --json | jq '.prospects[] | select(.score >= 8)'
```
