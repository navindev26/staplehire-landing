Get from zero to a working CLI session in a few minutes. For the full hiring loop (source → inbox → interview → hire), see the [CLI command reference](/cli/commands).

## Prerequisites

- Node.js 18+ and npm
- A Staplehire account

## Get started

<Steps>
  <Step title="Install">
    Install the CLI globally:

    ```bash
    npm install -g @staplehire/staplehire-cli
    ```
  </Step>
  <Step title="Log in">
    Authenticate with your Staplehire account (opens the browser):

    ```bash
    staplehire login
    ```
  </Step>
  <Step title="List roles">
    Confirm the session and inspect roles as JSON:

    ```bash
    staplehire roles list --json
    ```
  </Step>
</Steps>
