The Staplehire CLI (`staplehire`) exposes the same hiring loop as the Staplehire dashboard — roles, sourcing, inbox triage, candidate research, interviews, and outreach — as stable commands with **JSON output when piped** and human-readable output in an interactive shell.

Use it from your terminal or wire it into AI editors (Cursor, Claude Code, Codex, OpenClaw, Hermes, Pi) so agents can source, screen, interview, and hire without leaving your repo.

## What you can do

| Stage | CLI focus |
| --- | --- |
| **Source** | Create roles from a JD, start sourcing, list prospects |
| **Inbox** | List role inboxes, triage applications, surface strong fits |
| **Investigate** | Enrich candidates, run research, read screening results |
| **Interview** | Create interview designs, send invites, list sessions |
| **Hire** | Send agent mail and move candidates through the pipeline |

## Output modes

- **Interactive terminal** — formatted tables and prompts for humans.
- **Automation / agents** — append `--json` (or pipe stdout) for machine-readable records your agent can rank, filter, and act on.
