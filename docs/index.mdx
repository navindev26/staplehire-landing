Staplehire is an AI-native hiring platform: roles, sourcing, inbox triage, screening, interviews, and outreach — built for humans and for agents.

Use the **Staplehire CLI** from your terminal or editor to run the same hiring loop as the dashboard, with JSON output for automation.
