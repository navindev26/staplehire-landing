The Staplehire CLI (`staplehire`) runs the full hiring loop from your terminal or editor — **source, inbox, investigate, interview, hire** — with **JSON output** when piped and readable output in an interactive shell.

```bash
npm install -g @staplehire/staplehire-cli
staplehire login
staplehire roles list --json
```

## What you can do

| Stage | CLI focus |
| --- | --- |
| **Source** | Create roles from a JD, start sourcing, list prospects |
| **Inbox** | List role inboxes, triage applications, surface strong fits |
| **Investigate** | Enrich candidates, run research, read screening results |
| **Interview** | Create interview designs, send invites, list sessions |
| **Hire** | Send agent mail and move candidates through the pipeline |

## Output modes

- **Interactive terminal** — tables and prompts for humans.
- **Agents / scripts** — `--json` or pipe stdout for machine-readable records.
